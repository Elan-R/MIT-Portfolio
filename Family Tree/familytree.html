Choose a JSON file that holds family tree data and click "Build Tree".

The JSON should be a list of objects with the keys "name", "id", "partner", and "children".

The "id" and "partner" keys should have string values with other objects' ids and the "children" key should have a list of strings value of other objects' ids.

If multiple files are chosen, objects with equivalent "id" values will be treated as the same.
